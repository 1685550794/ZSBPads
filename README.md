抽抽面试助手，一款新闻阅读 App框架，基于 Material Design + MVP + OKHTTP  + Glide，基本涵盖了当前 Android 端开发最常用的主流框架，基于此框架可以快速开发一个app。

[安装地址](http://a.app.qq.com/o/simple.jsp?pkgname=com.lcc.msdq)
